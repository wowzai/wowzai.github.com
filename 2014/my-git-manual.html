git自我教程

首先，需要说明的是这是写给自己看的，所以叫自我教程，这里只是记录我经常要用的一些命令，并且会持续更新，由于我最近刚刚开始使用git，所以很多东西还不懂的使用。

初始化库：

git init

增加文件：

git add <filename>

提交文件：

git commit -m "comment"

查看远程库：

git remote -v

增加远程库：

git remote add <name> <url>

eg: git remote add wowzai git@github.com:wowzai/repo.git

如果发现远程库写错了，可以删除远程库：

git remote remove <name>

从远处库中拉取远程库到本地库：

git pull remote-repo-name

暂时我常用的一些git命令就是这些，对于像合并、建立分支我暂时还没用到，所以这里先不记录，等用到了再来了解。
