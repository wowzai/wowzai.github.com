JavaScript学习笔记

1.HTML是顺序加载的，如果改变界面元素的话，需要将script放在下面，这样才能保证script执行的时候操作界面元素可以找到对应的界面元素：

var myVariable = 'Bob';
myVariable = 'Steve';

疑问： ==与===的区别？

==在比较时如果两个变量的类型不同，会作类型转换，具体如何转换没有了解

 1   ==  1     // true
"1"  ==  1     // true
 1   == '1'    // true
 0   == false  // true

===是严格的比较，不同的类型也不会转换

3 === 3   // true
3 === '3' // false

同样，!=和!==的区别也一样

函数： 函数可以提高代码的复用率：

function multiply(num1,num2) {
    var result = num1 * num2;
    return result;
}

multiply(4,7);
multiply(20,20);
multiply(0.5,3);

在函数外面的定义的变量是全局变量，在函数里面定义的变量是局部变量，JavaScript中没有块语句域，即在花括号的变量中作用域也是在整个函数中可用的，而不是只能在花括号里能用：

if (true) {
  var x = 5;
}
console.log(x);  // 5

关于变量的作用域，可以阅读： JavaScript变量作用域

事件：

网页的交互就需要事件，例如点击事件：

document.querySelector('html').onclick = function() {
    alert('Ouch! Stop poking me!');
}

给整个html赋上Click事件

document.querySelector('html').onclick = function() {};

等价于：

var myHTML = document.querySelector('html');
myHTML.onclick = function() {};

如果在父子标签中都设置了事件，两个事件都会执行，例如在HTML赋事件后再在H1标签上赋事件，则点击H1文字后，先执行H1的事件，再执行HTML的事件：

document.querySelector('html').onclick = function() {
    alert('Ouch! Stop poking me!');
}

document.querySelector('h1').onclick = function() {
    alert('h1 click!');
}

点击H1的文字先弹出“h1 click”再弹出“Ouch! Stop poking me!” （chrome 41）
